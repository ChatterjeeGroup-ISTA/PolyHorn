get_UNSAT_constraint(need_strict=False) → List[CoefficientConstraint][source]ÔÉÅ

A function to find the constraints when it is not satisfiable.

Two set of constraint can be generated:

  1. LHS => -1 >= 0

  2. w_j^2*d = h_1*(g_1 - w_1^2) + h_2*(g_2 - w_2^2) ‚Ä¶ h_n*(g_n - w_n^2) for some j where g_j is strict

Parameters:
