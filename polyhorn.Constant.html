class polyhorn.Constant.AvailabilityDict(*args, **kwargs)[source]

Bases: dict

AvailabilityDict is a class which is used to define a dictionary that can be used to hierarchically check the availability of commands in the system, given an array of possible commands.

Parameters:

  • *args (Any) – The arguments that are passed to the dictionary.
