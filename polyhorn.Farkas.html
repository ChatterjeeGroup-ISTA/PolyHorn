get_UNSAT_constraint(need_strict: bool = False) → List[CoefficientConstraint][source]

A function to find the constraints when it is not satisfiable.

Two set of constraint can be generated:

  1. LHS => -1 >= 0

  2. LHS => 0 > 0

This function returns a polynomial y_0 + y_1*f_1 + y_2*f_2 …+ y_n*f_n where f_i are left hand side polynomials and y_i are newly created variable in farkas theorem.

Parameters:
