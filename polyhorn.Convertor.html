polyhorn.Convertor.convert_general_string_to_poly(poly_text: str, all_variables: List[UnknownVariable], program_variables: List[UnknownVariable]) → Polynomial[source]

This function convert a string to Polynomial.

It uses Lark to parse the string.

Parameters:

  • poly_text (str) – Text that should be converted to the Polynomial.

  • all_variables (List[UnknownVariable]) – List of all variables in the text.

  • program_variables (List[UnknownVariable]) – List of program variables.

Returns:

Polynomial that the input string represents.

Return type:

Polynomial
